A cosa serve bootsrap?

bootstrap è un editor gratuito per sviluppare più velocemente e più facilmente.

Cosa devi importare per creare una pagina web con bootstrap

è possibile includere bootsrap senza scaricarlo includendolo da un CDN

Cos’è un CDN? Che vantaggi comporta rispetto a scaricare bootstrap sul tuo sito?

Quando visiti un sito c'è un tempo di download più veloce. Quando richiedi un file da esso ti riserva il server più vicino, e ciò lo rende più veloce

A cosa serve questo attributo del tag meta?

Serve per far adattare il sito a qualsiasi dispositivo voglia visualizzarlo

Che differenza c’è tra Container e Container Fluid (prova il container fluid creando una pagina chiamata BTS_1.1.1.html)

Il container-fluid si espande per tutta la width della pagina, mentre il container normale ha una width fissa.
